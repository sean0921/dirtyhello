# Dirty Debian Packaged Hello World

Dirty packaged hello world written in C in Debian format

## Install Build Dependencies

```
sudo apt install build-essential dpkg-dev
```

## Build

```
./build.sh
```

## Install

```
sudo apt install ./dirtyhello_0.0.1_amd64.deb
```
